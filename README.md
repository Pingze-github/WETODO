# WETODO
一个简单的共享型待开发TODO列表，用于微团队协作。

## 说明
外网开发使用，置于174.138.*.148服务器

## 基本需要
+ 列表显示，简洁明了
+ 操作尽量精简，可以有快捷键（如shift+delete不询问立刻删除当前条目）
+ 条目应有简单分类，按优先程度（最优先、优先、无）和工种（前端、后端、策划）
+ 条目可以标记完成，置于后方
+ 列表应按天结算。只显示当天的
+ 前一天未完成的条目，应继续顺延至下一天
+ 可以查看历史记录
+ 有唯一密码登录功能，信息保密

## 进阶需要
+ 按ip等区分使用者，并可以为自己命名。条目上显示发起者名
+ 可以按工种导出为格式化字符串，用于填写工时
